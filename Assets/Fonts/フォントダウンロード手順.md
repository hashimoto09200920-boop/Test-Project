# 日本語フォント ダウンロード手順

Unity のスキルカード用に、日本語を多く含むフォントをダウンロードする手順です。  
**どちらか一方**を選んでください。

---

## 方法A: Noto Sans CJK JP（Google・推奨）

文字数が多く、Unity でよく使われます。

### 1. Releases ページを開く（ここが一覧です）

ブラウザのアドレス欄に **そのまま** 次の URL を入力して開く：

**https://github.com/notofonts/noto-cjk/releases**

※ リポジトリのトップや Serif フォルダには「Releases の数字」は出ないことがあります。上記 URL を直接開いてください。

### 2. 使いたいリリースを選ぶ

- ページには **リリースの一覧**（Serif2.003、Sans2.004 など）が並んでいます
- **Noto Sans CJK**（ゴシック）が欲しい → **「Noto Sans CJK Version 2.004」** のタイトルをクリック
- **Noto Serif CJK**（明朝）が欲しい → **「Noto Serif CJK Version 2.003」**（いちばん上、Latest）のタイトルをクリック

### 3. 日本語用の ZIP をダウンロード

- 開いたリリースの説明の下までスクロールすると **「Assets」** という欄がある
- その中の **日本語用** のリンクをクリックしてダウンロードする：
  - **Sans** の場合: 日本語用の ZIP（例: `NotoSansCJKjp` などと書いてあるもの）
  - **Serif** の場合: **「07_NotoSerifCJKjp.zip」**（Language Specific OTFs Japanese）

### 4. 解凍する

- ダウンロードした ZIP を右クリック → **「すべて展開」**（または「解凍」）
- 中に入っている **.ttf** または **.otf** を確認する

### 5. Unity のプロジェクトに入れる

- 解凍した **.ttf** または **.otf** を、次のフォルダに **コピー** する：  
  **`（プロジェクト）/Assets/Fonts/`**
- Unity エディタに戻ると自動でインポートされる

### 6. TextMeshPro の Font Asset を作る（別の手順で）

- **Window** → **TextMesh Pro** → **Font Asset Creator** を開く
- **Source Font** に、今コピーした **Noto Sans CJK JP** のフォントを指定
- **Character Set** を **Unicode Range (Hex)** にし、**Character Sequence** に  
  `20-7E,3000-303F,3040-309F,30A0-30FF,4E00-9FFF,FF00-FFEF` を入力
- **Atlas Resolution** を **2048 x 2048**（または 4096）に
- **Generate Font Atlas** を押す
- **Save as...** で **新しい名前**（例: `NotoSansCJKjp SDF`）で **Assets/Fonts/** に保存
- ゲームシーンで **SkillSelectionUI** の **Japanese Font Asset** に、その新しいアセットを指定

---

## 方法B: 源ノ角ゴシック（Adobe / Google）

商用利用可。Noto と同系統で、文字の収録数が多いです。

### 1. ダウンロードページを開く

ブラウザで次の URL を開く：

**https://github.com/adobe-fonts/source-han-sans/releases**

### 2. 最新のリリースを選ぶ

- **「Latest」** または一覧のいちばん上のリリースをクリックする

### 3. 日本語用の OTF をダウンロード

- **Assets** 一覧から、**日本語用**の ZIP を選ぶ：
  - **`SourceHanSans-J.zip`**（J = 日本語）  
  または  
  - **`SourceHanSans-VF.zip`**（可変フォント・1ファイルで複数ウェイト）

- クリックして ZIP をダウンロードする

### 4. 解凍する

- ダウンロードした ZIP を解凍する
- 中にある **.otf** ファイルを確認する（複数ある場合は **Regular** や **Normal** などが使いやすい）

### 5. Unity のプロジェクトに入れる

- 使いたい **.otf** を **`（プロジェクト）/Assets/Fonts/`** にコピーする
- Unity で自動インポートされる

### 6. TextMeshPro の Font Asset を作る

- 方法A の「6. TextMeshPro の Font Asset を作る」と同じ手順
- **Source Font** に今コピーした **源ノ角ゴシック** の .otf を指定し、**新しい名前**で SDF を保存してから、SkillSelectionUI の **Japanese Font Asset** に指定する

---

## 注意

- **既存の「NotoSansJP-Regular SDF」は上書きしない**でください。**Save as...** で必ず **別名**で保存してください。
- アトラス生成には数十秒〜数分かかることがあります。
- 不明な点があれば、このファイルの手順をひとつずつ確認しながら進めてください。
